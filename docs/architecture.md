# Architecture

Security Questionnaire Answer Studio is a static-friendly TypeScript executive-intelligence layer for the Kinetic Gain trust-evidence and questionnaire-answer estate.

- `src/data/sampleVerticalBrief.ts` holds the modeled buyers, answer families, and review routes.
- `src/analyze.ts` scores answer coverage, trust evidence, evidence linkage, response maturity, and turnaround drag.
- `src/services/verticalBriefService.ts` exposes the questionnaire-queue, answer-packs, trust-evidence, and cycle-time packets used by both the app and prerender step.
- `src/services/render.ts` renders the executive HTML surfaces and the sample JSON output.
- `scripts/prerender.ts` emits the static site, API payloads, `robots.txt`, and `sitemap.xml` for GitHub Pages.
